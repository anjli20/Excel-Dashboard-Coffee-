# Coffee Sales Dashboard — Excel Interactive Dashboard

An interactive Excel dashboard built to transform raw coffee sales transactional data
into clear, actionable business insights. The dashboard uses Pivot Tables, Pivot Charts,
slicers, and timeline controls to enable dynamic filtering and visual performance tracking
across time, product type, geography, and customer segments.

## Background

Modern organisations generate large volumes of transactional data, but raw data alone
does not support efficient strategic decision-making. This project addresses the absence
of a centralised, interactive reporting system for a coffee sales dataset containing
over 1,000 order records spanning multiple years, product categories, customer segments,
and geographic regions.

## Problem Statement

Without a structured analytical interface, extracting insights from the dataset requires
manual filtering, repeated calculations, and ad-hoc analysis — which is time-consuming
and error-prone. This project builds a single interactive dashboard to replace that
manual process.

## Objectives

- Monitor total sales performance over time by year and month
- Compare sales across different coffee types (Arabica, Excelsa, Liberica, Robusta)
- Compare sales across roast types (Dark, Light, Medium)
- Identify top-performing customers based on total revenue contribution
- Analyse sales distribution by country
- Evaluate the impact of customer loyalty card membership
- Enable dynamic filtering using slicers and timeline controls

## Tech Stack

| Tool | Purpose |
|---|---|
| Microsoft Excel | Dashboard, Pivot Tables, Pivot Charts, Slicers, Timeline |
| Excel Formulas | VLOOKUP / INDEX-MATCH for data enrichment |
